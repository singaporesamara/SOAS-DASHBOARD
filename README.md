# SOAS Client App

## Installation

```js
$ yarn
$ yarn start # to start dev application
$ yarn run lint:watch # to run eslint in watch mode
```

## Deployment

* ```ps ax | grep node```
* ```kill -9 NODE_PID```
* ```./start.sh```
* browse http://dev-soaspay.com:8000/
